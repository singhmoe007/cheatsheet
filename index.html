What is HTML?

HTML stands for "HyperText Markup Language". It is the language used to create websites. It is like a set of instructions that tells your web browser (like Google Chrome or Safari) how to display web pages on your computer or mobile device.

Semantic HTML:Semantic HTML refers to the use of HTML elements that convey meaning beyond just formatting. This helps both search engines and assistive technologies better understand the structure and purpose of a webpage.

Semantic HTML Elements

Tag Name Description
<article> Article Respresents a self contained part of page, i.e blog etc.
<aside> Aside Shows additional info about current content.
<details> Details Encapsulates the <summary> element and any additional details.
<figcaption> Figcaption Describes the media encapsulated within the <figure> element.
<figure> Figure Encapsulates media such as an image, diagram, or code snippet.
<footer> Footer Represents end of the page, displays social media links etc.
<header> Header Represents introductory part of page.
<main> Main Represents the primary content within the body element.
<mark> Mark Represents a part that will appear highlighted.
<nav> Nav Defines a block of navigation links such as menus and tables of contents.
<section> Section Defines elements in a document.
<summary> Summary This element reveals additional info when clicked.
<time> Time Represents a time-related piece of information.
